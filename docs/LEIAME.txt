    DOCSANALYSER v2.0 - Sistema de Análise de Documentos E-DIPLOMA
═══════════════════════════════════════════════════════════════════

📋 O QUE É?
───────────────────────────────────────────────────────────────────
O DocsAnalyser é um sistema inteligente que:
  • Lê automaticamente documentos (PDF, DOCX, TXT)
  • Identifica o tipo de documento (Ofício, Termo, Identidade, etc)
  • Encontra o nome do estudante e o RA (matrícula)
  • Renomeia os arquivos no padrão E-DIPLOMA correto

═══════════════════════════════════════════════════════════════════

🚀 COMO USAR (VERSÃO EXECUTÁVEL)
───────────────────────────────────────────────────────────────────
1. Execute o arquivo: DocsAnalyser.exe

2. Na tela que abrir:
   ✓ Clique em "Selecionar Diretório"
   ✓ Escolha a pasta com seus documentos
   ✓ Clique em "Processar Documentos"
   ✓ Aguarde a conclusão

3. Pronto! Os arquivos serão renomeados automaticamente

⚠️ IMPORTANTE
───────────────────────────────────────────────────────────────────
• Por padrão, o sistema cria backups dos arquivos originais
• Os nomes dos arquivos devem seguir algum padrão reconhecível
• Para PDFs escaneados (imagens), configure a chave do Gemini AI

🎯 CÓDIGOS DE DOCUMENTOS
───────────────────────────────────────────────────────────────────
OFI = Ofício de Solicitação
TER = Termo de Responsabilidade
IDE = Documento de Identidade
ENS = Certificado de Ensino Médio
HIS = Histórico Escolar
REC = Recibo de Pagamento
CER = Certidão (Nascimento/Casamento)
